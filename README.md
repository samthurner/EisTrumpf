## **Beschreibung**
Diese Anwendung ist eine grafische Umsetzung des klassischen **Quartett-Kartenspiels**, wie wir es aus unserer Kindheit kennen.  
Als Spieler tritt man in einer JavaFX Anwendung gegen einen anderen Spieler an!  
Die App unterstützt **eigene Decks**, die als JSON-Datei importiert und gespeichert werden können.  
Benutzerdaten wie das **ausgewählte Deck**, aber auch eine **Winstreak** werden lokal in einer JSON-Datei mit Hilfe von Gson gespeichert, sodass die Einstellungen und Statistiken auch nach einem Neustart erhalten bleiben.
**Gespielt wird in der IDE**

---

## Spielprinzip
Man Spielt im **1 gegen 1** gegen einen Freund.
Die Spieler wählen **abwechselnd** Statistiken auf, welche Sie mit der Karte des Gegners **vergleichen**. Der Spieler mit dem höheren Wert **bekommt beide soeben miteinander verglichenen Karten**.

Das Spiel endet, wenn ein Spieler **alle Karten** besitzt — wer alle Karten hat, gewinnt!

## Kartendecks
#### Mitgelieferte Kartendecks:
Es gibt **4 Standartkartendecks**, die direkt mitgeliefert werden und Spielbereit sind.
1. Tier-Deck
2. Auto-Deck
3. Fußballerdeck
4. HTL Lehrer
#### Eigene Kartendecks:
Man kann auch selbst **Kartendecks** hochladen, muss diese aber schon davor in einer passenden **Json-Struktur** erstellen.
Eine **Vorlage**, wie ein so ein **Kartendeck** aussehen soll gibt es auch im Repo: `Vorlage eines Kartendecks.md`
